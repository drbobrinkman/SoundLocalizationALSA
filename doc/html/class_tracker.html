Detailed Description

Manages a collection of Trackable, including clustering nearby sounds and tracking duration of sounds

Note
Singleton, with lazy initialization. (Meyers style singleton)
Bug:
We do not gracefully handle wrap-around or overflow of the frame numbers. Overflow should take 9 billion years, so we ought to be okay.
